
#list of command i have used so far

git inti - It is used to intiliaze the git.

git status - to check the status of the branch (commited , untracked, staged).

./git - this folder holds the full git config files like index/log/config/objects.

git branch - to check on which branch you are.

git log- to check the commits logs.

git log --oneline - to check the logs in one short line.

git diff - it is used to check the diff made in file after commit happened.

git diff --staged - it is used to check the diff made after being staged.
